# Welcome to SSS Core Management System

## Project Overview

SSS Core is a premium HR and Inventory management system designed for executive-level tracking and reporting. It features a sophisticated Golden & Teal aesthetic, providing a high-end interface for monitoring attendance, human resources, and warehouse inventory.

## How to use this project locally

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd sss-core-hrms

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies Used

This project is built with a modern stack for high performance and premium UI/UX:

- **Frontend**: Vite, React, TypeScript
- **Styling**: Tailwind CSS, Lucide React (Icons), shadcn-ui
- **Database/Backend**: Supabase
- **Analytics**: Recharts
- **Theme**: Custom Golden & Teal Executive Theme

## Key Features

- **Executive Dashboard**: Real-time KPI tracking for employees, attendance, and departments.
- **Attendance Tracker**: Geo-location and photo capture enabled attendance logging.
- **Inventory Management**: Warehouse tracking with "Golden Formula" stock calculations.
- **Labour Management**: Site-specific contractor and manpower reporting.
- **Premium Design**: Sophisticated glassmorphism and animated UI elements.
